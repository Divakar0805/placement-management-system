# Placement Management System

A Full Stack Placement Management System developed using Java, Spring Boot, MySQL, HTML, CSS, and JavaScript.

---

## Features

### Student Module
- Student Registration
- Student Login
- Update Profile
- View Companies
- Apply for Companies
- View Applied Companies
- Placement Status

### Admin Module
- Admin Login
- Manage Students
- Manage Companies
- View Applications
- Search Applications
- Update Placement Status

---

## Technologies Used

- Java
- Spring Boot
- MySQL
- HTML
- CSS
- JavaScript
- Git & GitHub

## How to Run

1. Clone the repository

```bash
git clone https://github.com/Divakar0805/placement-management-system.git
```

2. Open project in IntelliJ IDEA

3. Configure MySQL database

4. Run Spring Boot Application

5. Open browser:

```text
http://localhost:8081
```
